# Samsung Galaxy J2 Prime (SM-G532F) Custom TWRP Release

This project is a visually and systemically customized, Turkish-focused version of the TWRP recovery image built for the Samsung Galaxy J2 Prime.

## 🚀 Features
- **Custom Splash Screen:** Added a custom developer-signed boot logo tailored to the device's native resolution ($540 \times 960$).
- **Extended Splash Duration:** Increased the splash screen timeout via configuration to ensure the signature is easily readable.
- **Language Optimization:** Removed unnecessary language files and optimized the default interface language to Turkish.

## 🛠️ Installation
1. Transfer the `image-new.img` file to your device's internal storage or SD card.
2. Boot into your current TWRP recovery and navigate to `Install` -> `Install Image`.
3. Select the `image-new.img` file from your storage.
4. Choose **Recovery** as the target partition and swipe to confirm the flash.
